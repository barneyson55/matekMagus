# Háttér Opciók

A felhasználók választhatnak különböző háttereket a program személyre szabásához.

## Cél

- Vizuális változatosság.
- Kényelmes tanulási környezet.

## Háttér Típusok

### 1. Egyszínű
- Fehér (#ffffff)
- Sötét (#121212)
- Kék (#e3f2fd)

### 2. Mintás
- Finom rács (matematikai érzés).
- Pontminta (minimalista).
- Geometriai alakzatok (körök, négyzetek).

### 3. Képalapú
- Absztrakt matematikai minták (pl. függvénygrafikonok).
- Természetes színek (erdei, óceáni).

## Megvalósítás

- CSS background tulajdonságok.
- Felhasználói menüben választás.
- Opcionális animált hátterek (pl. lassú mozgás).
