Exponenciális és logaritmus függvények

Rövid bevezetés (Mi ez? Mire jó?)

Az exponenciális függvény gyors növekedést vagy gyors csökkenést ír le, például kamatos kamatnál vagy népességnél. A logaritmus ennek a visszafordítója, vagyis megmondja, milyen kitevő kell egy adott eredményhez. A két függvény egymás inverze, ezért a grafikonjuk tükrözésben áll az y = x egyenesre. Ha ezt megérted, sok képlet egyszerűsödik.

Alapfogalmak / definíciók

  • Exponenciális alak: f(x) = a^x, ahol a > 0 és a ≠ 1.
  • Logaritmus alak: g(x) = log_a x, ahol x > 0.
  • Inverz kapcsolat: log_a b = x ⇔ a^x = b.
  • Tartomány: exponenciálisnál R, logaritmusnál (0, ∞).
  • Értékkészlet: exponenciálisnál (0, ∞), logaritmusnál R.
  • Metszéspont: a^0 = 1 és log_a 1 = 0.
  • Aszimptóta: exponenciálisnál y = 0, logaritmusnál x = 0.

Lépésről lépésre példa (1–2 db)

1. példa – exponenciális egyenlet

Oldd meg: 3^x = 81.

  1. Írd át: 81 = 3^4.
  2. Azonos alapnál a kitevők egyenlők: x = 4.

2. példa – logaritmus értéke

Számold ki: log_2(1/8).

  1. Írd át exponenciálisan: 2^x = 1/8.
  2. 1/8 = 2^{-3}, ezért x = -3.

Vizuális magyarázat (inverz párok)

x 2^x log_2(x)
0 1 –
1 2 0
2 4 1
3 8 log_2(8) = 3

A logaritmus azt mondja meg, hogy mennyi a kitevő.

Gyakori hibák / tévhitek

  • Elfelejted, hogy logaritmusnál x > 0.
  • Megengeded az a = 1 vagy a ≤ 0 alapot.
  • Összekevered a log_a b és a log_b a jelentését.
  • Nem írod át exponenciális alakba a feladatot.
  • Elfelejted, hogy az exponenciális függvény mindig pozitív.

Mini önellenőrzés (válasz nélkül)

  • Mi a tartománya a f(x) = 3^x függvénynek?
  • Számold ki: log_5(25).
  • Oldd meg: 2^x = 1/4.
  • Hol metszi a logaritmus függvény az x-tengelyt?
  • Mi történik a függvénnyel, ha 0 < a < 1?

Összefoglaló

  • Az exponenciális és a logaritmus függvények egymás inverzei.
  • f(x) = a^x mindig pozitív értékeket ad.
  • g(x) = log_a x csak pozitív x-re értelmezett.
  • Az aszimptóta exponenciálisnál y = 0, logaritmusnál x = 0.
  • Feladatnál gyakran segít az átírás: log_a b = x ⇔ a^x = b.

Teszt

Exponenciális és logaritmus függvények: alapértékek, inverz kapcsolat, tartomány és monotonitás.

Megoldási tippek

  • Mindig ellenőrizd: a^x = b vagy loga b = x.
  • Az exponenciális függvény értékkészlete mindig pozitív.
  • Ha 0 < a < 1, akkor a függvény csökken.
