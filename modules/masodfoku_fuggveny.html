Másodfokú függvény

Rövid bevezetés (Mi ez? Mire jó?)

A másodfokú függvény grafikonja parabola, ami sok mozgásos és optimalizálási helyzetben előjön. Ilyen például egy labda röppályája vagy egy terület maximalizálása. A három együttható (a, b, c) megmutatja, merre nyílik a parabola és hol van a csúcsa. Ha ezeket gyorsan leolvasod, könnyebben rajzolsz grafikont és oldasz feladatot.

Alapfogalmak / definíciók

  • Általános alak: f(x) = ax^2 + bx + c, ahol a ≠ 0.
  • Csúcs x-koordinátája: x_v = -b / (2a).
  • Csúcs y-koordinátája: y_v = f(x_v).
  • Tengely: x = x_v (szimmetriatengely).
  • Nyitottság: a > 0 felfelé, a < 0 lefelé.
  • Diszkrimináns: D = b^2 - 4ac (zérushelyek száma).
  • Zérushelyek: D > 0 két, D = 0 egy, D < 0 nincs valós.
  • Y-metszet: f(0) = c.

Lépésről lépésre példa (1–2 db)

1. példa – csúcs és tengely

Legyen f(x) = 2x^2 - 4x - 6.

  1. Olvasd le: a = 2, b = -4, c = -6.
  2. x_v = -b/(2a) = 4/4 = 1.
  3. y_v = f(1) = 2 - 4 - 6 = -8, tehát a csúcs (1, -8).
  4. Mivel a > 0, felfelé nyílik (minimum a csúcsban).
  5. Y-metszet: (0, -6).

2. példa – csúcsalakból

Legyen f(x) = -(x - 2)^2 + 3.

  1. A csúcs: (2, 3).
  2. a = -1, ezért lefelé nyílik (maximum 3).
  3. A tengely: x = 2.
  4. Plusz pont: f(0) = -4 + 3 = -1.

Vizuális magyarázat (csúcs és tengely)

A parabola szimmetrikus a tengelyre, a csúcs a minimum vagy maximum pont.

Gyakori hibák / tévhitek

  • Elfelejted, hogy a ≠ 0.
  • Hibás előjelet írsz a -b/(2a) képletben.
  • Összekevered a csúcs koordinátáit a tengely egyenletével.
  • Úgy gondolod, hogy D < 0 esetén is van zérushely.
  • Elfelejted leolvasni az y-metszetet (c).

Mini önellenőrzés (válasz nélkül)

  • Mi a tengely és a csúcs a f(x) = x^2 + 4x + 1 függvénynél?
  • Felfelé vagy lefelé nyílik a f(x) = -2x^2 + 3x + 1 parabola?
  • Mennyi a y-metszet a f(x) = 3x^2 - 7 függvénynél?
  • Hány zérushely van, ha D < 0?
  • Mi a csúcs a f(x) = 2(x - 3)^2 + 1 alaknál?

Összefoglaló

  • A másodfokú függvény alakja f(x) = ax^2 + bx + c.
  • A csúcs: x_v = -b/(2a), y_v = f(x_v).
  • a előjele dönti el a nyitottságot.
  • A diszkrimináns mutatja a zérushelyek számát.
  • Y-metszet: f(0) = c.

Vizuális modell

Állítsd be az a, b, c együtthatókat, és figyeld meg a parabola fő tulajdonságait.
