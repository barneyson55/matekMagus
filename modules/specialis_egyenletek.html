Speciális egyenletek

Rövid bevezetés (Mi ez? Mire jó?)

Speciális egyenleteknél abszolútérték, gyök vagy tört is szerepel a kifejezésben. Itt nem elég csak „rendezni”, hanem a feltételeket is figyelembe kell venni. Az abszolútértéknél eseteket bontunk, gyökös egyenletnél pedig ellenőrizzük a definíciós tartományt. Mindig szükséges a visszaellenőrzés, mert könnyen adódhat hamis gyök.

Alapfogalmak / definíciók

  • Abszolútérték: |x − a| a távolság a-tól a számegyenesen.
  • Esetbontás: |x − a| = b → x − a = b vagy x − a = −b.
  • Gyökös egyenlet: √(x + a) = b, ahol b ≥ 0.
  • Definíciós tartomány: a megengedett x-ek (pl. x + a ≥ 0).
  • Tört egyenlet: nevező nem lehet 0 (pl. x ≠ a).
  • Hamis gyök: a műveletek után kapott, de visszaellenőrzéskor kieső megoldás.

Lépésről lépésre példa (1–2 db)

1. példa – abszolútérték

|x − 3| = 5 megoldása:

  1. x − 3 = 5 → x = 8.
  2. x − 3 = −5 → x = −2.
  3. Mindkettő jó, mert b = 5 ≥ 0.

2. példa – gyökös egyenlet

Oldd meg: √(x + 1) = 4.

  1. Feltétel: x + 1 ≥ 0.
  2. Négyzetre emelés: x + 1 = 16.
  3. x = 15, ami megfelel a feltételnek.

Vizuális magyarázat (távolság a számegyenesen)

A pontok 5 egységre vannak a 3-tól.

Gyakori hibák / tévhitek

  • b < 0 esetén is próbálsz abszolútértéket megoldani.
  • Gyökös egyenletnél nem ellenőrzöd a definíciós tartományt.
  • Négyzetre emelés után nem ellenőrzöl, így hamis gyök marad.
  • Tört egyenletnél nem zárod ki a nevező 0-ját.
  • Elfelejted mindkét esetet felírni abszolútértéknél.

Mini önellenőrzés (válasz nélkül)

  • Oldd meg: |x + 2| = 7.
  • Oldd meg: √(x − 3) = 2.
  • Melyik x érték tiltott a 1/(x − 4) = 2 egyenletnél?
  • Miért kell visszaellenőrizni négyzetre emelés után?

Összefoglaló

  • Speciális egyenleteknél először a feltételeket nézd.
  • Abszolútérték: két eset, gyökös: tartomány + négyzetre emelés.
  • Tört egyenletnél zárd ki a nevező 0-ját.
  • Mindig ellenőrizd, hogy nincs hamis gyök.

Teszt

Oldd meg a speciális egyenleteket, és döntsd el a megoldások számát.

Megoldási lépések

  • Abszolútértéknél bonts eseteket.
  • Gyökös egyenletnél nézd meg a definíciós tartományt.
  • Tört egyenletnél kerüld a tiltott értékeket.
