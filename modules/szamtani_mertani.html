Számtani és mértani sorozatok

A sorozatoknál a következő tag és az összeg képlete adja a gyors számolás alapját. Emelt szinten különösen fontos a képletek biztos használata.

Alapképletek

  • a_n = a_1 + (n-1)d (számtani)
  • S_n = n/2 (2a_1 + (n-1)d)
  • a_n = a_1 · q^{n-1} (mértani)
  • S_n = a_1 (q^n - 1)/(q-1), ha q ≠ 1

Számtani sorozat

  • Különbség: d minden tag között azonos.
  • Ha d > 0, akkor a sorozat növekvő.
  • Összeghez mindig használd a zárójel helyes számolását.

Mértani sorozat

  • Hányados: q minden tag között azonos.
  • Ha |q| < 1, a sorozat 0-hoz tart.
  • Az összegképlet csak q ≠ 1 esetén érvényes.

Példák

  • a1 = 3, d = 2, n = 5 → a5 = 11
  • a1 = 2, q = 3, n = 4 → a4 = 54
  • a1 = 1, q = 2, n = 3 → S3 = 7

Teszt

Megoldási tippek

  • Először írd fel a megfelelő képletet.
  • Ha törted van, egyszerűsíts, majd számolj.
  • Enterrel is továbbléphetsz.
Nehézség:
Számolásos feladat, rövid szám választ várunk.

Gyakorlás

Végtelenített gyakorlás. XP jutalom: könnyű = 1 XP, normál = 2 XP, nehéz = 3 XP.

Gyakorlási tipp

  • Gyorsan döntsd el: számtani vagy mértani sorozat?
  • Az n-edik tag és az összeg képlete eltér, figyelj a jelölésekre.
  • Helyes válasz után automatikusan jön a következő feladat.
